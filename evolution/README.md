# AI军团进化实录 / Evolution Chronicles

> "错误不被自己发现，就会被Stella审计发现，或被创始人发现。三选一——自己发现是代价最小的。" — 基因五·公开自批判节律

## 这是什么？

本目录记录AI军团每个Agent的实战反思与进化轨迹。不是美化后的"最佳实践"，而是真实的错误、发现、纠正和跃迁。

## 目录结构

```
evolution/
├── README.md                           # 本文件
├── tristan-reflection-YYYY-MM-DD.md    # Tristan的反思
├── stella-reflection-YYYY-MM-DD.md     # Stella的反思
├── shuyu-reflection-YYYY-MM-DD.md      # Shuyu的反思
├── ...                                 # 其他Agent
```

## 提交原则

1. **真实优先** — 暴露错误比包装完美更有价值
2. **案例驱动** — 每个观点必须附带具体时间+事件+数据
3. **跨Agent可见** — 一个Agent的教训=全军的免疫接种
4. **定期更新** — 重大事件后48h内、或每周至少一次

## 创始人铁律

> "已完成"的唯一定义：端口真实可连、数据真实可查、服务真实在运行、存证链真实可验证。缺任何一个，就不是已完成。
> 
> — 基因三·已完成检验
