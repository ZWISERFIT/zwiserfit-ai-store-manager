# zwiserfit AI Store Manager

AI店铺管理工具，集成OpenClaw智能代理，支持店铺运营自动化。

## 功能特性

- 🏪 店铺数据同步与管理
- 🤖 AI代理调度（基于OpenClaw）
- 📊 运营数据分析
- 🔄 自动化任务执行
- 📱 多平台集成

## 快速开始

详细部署指南请参阅 [docs/deployment.md](docs/deployment.md)。

### 环境要求
- Node.js 18+
- OpenClaw 环境（可选）
- GitHub Token（用于版本控制）

### 安装步骤
```bash
git clone https://github.com/ZWISERFIT/zwiserfit-ai-store-manager.git
cd zwiserfit-ai-store-manager
# 根据 deployment.md 完成配置
```

## 配置文件

### AGENTS.md
OpenClaw代理配置文件，定义AI代理的行为和职责。

### SOUL.md
数字分身身份定义，维护Shuyu总指挥的核心身份与使命。
